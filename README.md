#### Laravel API + SPA vue, vuex, router, axios ####
* Laravel v.7.0
* Laravel Passport v.9.3
* Vue v.2.6.11
* Vuex v.3.5.1

**Развертывание**
1. composer install
2. php artisan serve
2. cd frontend & yarn
3. yarn serve

Если будет отличаться адрес на dev, то поменять
frontend/.env.develop

Собирать frontend с папки frontend.  
Автоматически кладет куда нужно.

Для dev: yarn serve,  
для prod: yarn build

`Документация`:
api/documentation
